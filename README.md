# HelixFrontend-v2

- make sure node.js is installed
````bash
node -v
````

---

if node is installed, then install pnpm
````bash
npm install -g pnpm
````

---

close and reopen your terminal, then check:
````bash
pnpm -v
````

---

insall the project dependencies
````bash
pnpm install
````

---

run the project
````bash
pnpm run dev
```
